# .jscpd.json

## Purpose

Configuration for [jscpd](https://github.com/kucherenko/jscpd), the
copy/paste detector. Fails the build when duplicated code blocks exceed the
threshold, keeping the starter's small surface free of accidental copy-paste.

## Settings

- `threshold: 0.5` — max 0.5% duplication allowed before a non-zero exit.
- `minTokens: 50` — minimum token run length counted as a clone.
- `files: ["src/**/*.ts"]` — production + test TypeScript sources.
- `ignore` — vendor/build artefacts, declaration files, and test/spec files
  (test boilerplate legitimately repeats setup).

## Usage

```bash
pnpm run check:dup   # jscpd --config config/.jscpd.json src
```

## Tests

No unit test; exercised by the `check:dup` gate inside `pnpm test`.
