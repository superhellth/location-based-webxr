ℹ️ What is this app?

Purpose: Record AR + GPS data to test location alignment algorithms. Walk around while the app captures your path, GPS readings, and camera frames.

Scenario: A named physical area you'll revisit for multiple recordings (e.g., "Paris Eiffeltower"). Scenarios group related sessions together.

Session: A single recording within a scenario, automatically named by timestamp. Each session captures one walk-through.

Reference Points: Landmarks you mark during recording for ground-truth validation. Tap "📍 Mark Point" when standing at a known location.

Choose where to save this recording. Importing a previous recordings folder is optional — it's only needed to recover reference points the browser no longer has.
